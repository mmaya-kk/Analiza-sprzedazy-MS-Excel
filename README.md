Projekt wykonany w ramach studiów w programie MS Excel wraz ze sprawozdaniem wykonanym w LaTeX.

W projekcie skupiono się na pełnej automatyzacji raportu, aby uniknąć ręcznego wpisywania danych:

1.  **Dynamiczne nagłówki (MONTH1):** Zastosowano autorski mechanizm odmiany miesięcy przez przypadki (dopełniacz) przy użyciu tabeli pomocniczej i funkcji `WYSZUKAJ.PIONOWO`. Dzięki temu raport wyświetla np. "Wyniki dla miesiąca: **marca**" zamiast "marzec".
2.  **Zaawansowane wyszukiwanie:** Wykorzystanie duetu funkcji `INDEKS` oraz `PODAJ.POZYCJĘ` do precyzyjnego wyciągania danych o produktach z największą/najmniejszą sprzedażą.
3.  **Wykresy dynamiczne:** Automatyczna aktualizacja wizualizacji danych po zmianie parametrów wejściowych.
4.   **Sprawozdanie:** Profesjonalne sprawozdanie projektu wykonane w programie LaTeX -dokładnie opisuje przebieg projektu krok-po-kroku.
